Данный проект позволяет создать и работать с базой данных "Студенты". С помощью представленной утилиты можно осуществить учет обучающихся, составлять информационные справки. Программа состоит из нескольких модулей: два основных и несколько модулей-спутников, реализующие функционал.
Вы можете: как рядовой пользователь - добавлять записи и составлять справки; как пользователь-администратор: добавлять, удалять, редактировать записи, составлять справки и управлять правами других пользователей.
